# Telco_CustomerChurn_project
End-to-End Customer Churn Prediction Project: Data Cleaning, EDA, Machine Learning Models, Streamlit App, and Insights
Streamlit App Link: ("https://telcocustomerchurnproject.streamlit.app/")

## 🔹 Overview
This project predicts customer churn for a telecom company using machine learning. It includes:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Model Building & Comparison (Logistic Regression, Decision Tree, Random Forest)
- Model Evaluation
- Streamlit App for Interactive Predictions
- Business Insights & Recommendations

---

## 🔹 Objective
- Identify factors causing churn.
- Build a predictive model.
- Provide actionable insights to reduce churn.

---

## 🔹 Dataset
- Source: [Kaggle - Telco Customer Churn](https://www.kaggle.com/blastchar/telco-customer-churn)
- Features: Customer demographics, account info, services, and churn status.

---

## 🔹 Project Steps

1. **Data Cleaning**
   - Handle missing values, convert data types, encode categorical variables.

2. **EDA**
   - Visualize churn distribution, monthly charges vs churn, tenure impact, and correlations.

3. **Preprocessing**
   - One-hot encoding, train-test split, scaling features for certain models.

4. **Model Building**
   - Logistic Regression, Decision Tree, Random Forest.
   - Compare accuracy to select the best model.

5. **Model Evaluation**
   - Accuracy, classification report, confusion matrix.
   - Random Forest selected as final model.

6. **Streamlit App**
   - Interactive prediction using input features.
   - Shows churn probability and simple business insight.
   - Link: ("https://telcocustomerchurnproject.streamlit.app/")

---

## 🔹 Key Insights
- Customers with low tenure and month-to-month contracts churn more.
- Higher monthly charges increase churn probability.
- Long-term contracts improve retention.

---

## 🔹 Recommendations
- Offer promotions for new customers.
- Convert month-to-month to annual plans.
- Monitor high monthly charges to prevent churn.
